Choosing Fonts in CSS

How to Pick Fonts

Choosing the good font is very important for creating a website's experience. The difference between a good font and a better font can help create an identity for the brand and become a recognizable attribute. When picking a font, choose a font that is both easy to read and pleasing to look at. Specific fonts can help get different ideas across. Adjusting the emphasis, size, and color of text allows the creator to make numberless options for designs.

Using fonts with CSS

Font Family and Fallback Fonts

To choose the type of font you want, use the font-family property in CSS followed by the name of the font in quotation marks. If you are using a font that you downloaded there is a chance that it is not web safe and it is best to add a fallback font.

Font Pairing

Font pairing is an important piece of choosing fonts because it greatly adds to the aesthetics of a site. There are a few fundamental rules to help you choose multiple fonts.

1. Find complimentary fonts that are not too similar or different.

2. Utilize font superfamilies (fonts designed to work together).

3. Use fonts with contrast, like a serif with sans serif font.

4. Choose one main font to establish hierarchy on the page.
